# Portfolio

Portfolio web dynamique (V2) base sur les projets IMMO ETL et CRYPTO Ingest.

## Fonctionnalites

- Navigation sticky + barre de progression de lecture
- Recherche et filtres sur les projets
- Switch langue FR/EN (homepage)
- Timeline interactive avec filtres (type + annee)
- Formulaire de contact connecte (FormSubmit)
- Accessibilite renforcee (skip-link, focus visible, aria)
- Analytics privacy-friendly via Plausible
- Transitions de page (View Transitions API)
- Pipeline interactif anime
- Section "Proof of quality"
- Pages case-study detaillees:
  - `/projects/immo.html`
  - `/projects/crypto.html`
